What is Sentiment Analyst?

The Sentiment Analyst is a web application built with Flask, a Python web framework, designed to analyze the sentiment of text input provided by the user. The tool utilizes natural language processing techniques to extract meaningful insights from the text and determine the sentiment expressed within it.

How to Use it

To use the Sentiment Analyst, simply enter the text you would like to analyze in the text box provided and click the "Analyze Sentiment" button. The tool will process the text and provide you with a sentiment analysis, including the sentiment probabilities for each emotion category.
